# 新发发订餐系统

该仓库包含：
- 后端：Vercel Serverless Functions + MongoDB Atlas
- 前端：React SPA（移动端优先）

## 目录
- backend：API 与数据模型
- frontend：用户端/管理端前端

## 环境变量
在 backend 下配置 MONGODB_URI。

## 本地开发
- 前端：在 frontend 目录安装依赖并启动开发服务器
- 后端：在 Vercel 环境部署或本地使用 Vercel CLI 运行

## 功能覆盖
- 顾客点餐：桌号输入、菜单浏览、购物车、提交订单
- 管理后台：登录、订单管理、菜单管理、清空订单
